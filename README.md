알람 예제

### Background 작업
 - AlarmManager를 사용

### AlarmManager
 - RealTime(실제시간)으로 실행시키는 방법
 - Elapsed Time(기기 부팅된지 얼마나 지났는지)로 실행시키는 방법 : 좀 더 정확함
 
### 알람앱
지정된 시간에 알람이 울리게 할 수 있음(노티피케이션 이용)
지정된 시간 이후에는 취소하지 않는 한 매일 같은 시간에 반복되게 알람이 울리게 됨
